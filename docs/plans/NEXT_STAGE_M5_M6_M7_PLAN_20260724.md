## P0：完成 M5 冻结基线

完成条件：

1. 恰好 500 个唯一回合编号；
2. 几何、光度、标定、遮挡、组合五层各 100 回合；
3. 协议 SHA-256、Git 提交、运行环境和全部参数一致；
4. 无 GLFW/WGL 上下文错误；
5. 生成 JSON、CSV、分层汇总和失败回合种子清单；
6. 人工复跑每层至少 2 个成功与 2 个失败种子。

决策规则：

- 若名义回归失败，M5 无效，先修传统控制；
- 若目标获取率低，优先回到 M1/M2 改善传统/YOLO 粗检测；
- 若获取正常但深度/姿态门失败，优先修 M3 感知与时序估计；
- 若对准授权正常但插入撤权或接触，优先修 M4 安全门与轨迹；
- 不允许用 RL 掩盖系统性的感知标定错误。

## P1：M5.1 传统感知修复（按结果触发）

当前最可能的瓶颈是薄环轮廓不完整导致单目深度偏差。候选方案按风险从低到高：

1. 内外环联合尺度与轮廓完整度置信度；
2. 5–10 帧鲁棒中值/卡尔曼时序滤波；
3. 椭圆圆锥/PnP 姿态与深度联合估计；
4. 已知 TCP 微运动产生主动尺度观测；
5. YOLO 只接管远场发现和 ROI，不参与最终精度授权。

修复后必须使用新的协议指纹重跑相同 500 个种子，并与原始传统基线配对比较。

## P2：M6 有界残差强化学习

### 动作

```text
最终任务空间动作 = 冻结传统视觉伺服动作 + 安全门限幅后的 RL 残差
```

首版只允许小范围任务空间残差，不直接输出无约束六关节命令：

- 横向残差；
- 轴向步长调制；
- 两轴姿态残差；
- `hold / reobserve / fallback-to-coarse` 离散选择。

### 观测

只包含可部署信息：

- 外环/内环中心、直径、长短轴比和拟合误差；
- 检测置信度、连续有效帧数、目标丢失历史；
- 六关节状态、传统动作、上一残差和当前阶段；
- 不包含 MuJoCo 戳卡真值、真实间隙或真实轴角。

### 奖励与安全

按词典序组织，而不是简单加权后让效率压过安全：

1. 接触、过插、越界、无授权运动为最高惩罚；
2. 目标可见、质量门通过和最终对准；
3. 收敛时间、TCP 路径、折返、超调和抖动；
4. 残差幅度与动作变化率。

安全盾始终可撤销 RL：

- 视觉质量不足立即 `hold`；
- 超过残差界限裁剪；
- 任一安全裕度不足退回传统控制或粗对准；
- 插入授权仍由冻结的连续 5 帧门产生。

### 训练与验证

- 服务器用于训练和大规模仿真；
- 训练、调参、验证、冻结测试使用互不重叠的种子；
- EGL 像素训练域必须单独标定，不能直接引用 WGL 像素阈值；
- 最终对比至少包括传统基线、残差 RL、无安全盾消融和 RL-only 研究消融；
- RL-only 不进入安全演示主链。

M6 晋级条件：

- 安全指标不得低于冻结传统基线；
- 全流程成功率不下降；
- 路径、时间、折返或抖动至少一项在配对统计中稳定改善；
- WGL 与 EGL/域随机化测试均报告，不合并成单一数字。
