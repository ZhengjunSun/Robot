# 视觉对准与戳卡插入实施计划

## 执行原则

本计划自 2026-07-23 起作为项目正式开发主线。控制器在仿真和部署评估中只能读取：

- eye-in-hand 相机图像；
- 机器人关节、TCP 和允许部署的运行状态；
- 由视觉算法计算出的检测、几何和质量指标。

MuJoCo 中的戳卡真值、理想位姿和接触真值只能供评估器、数据标注或训练期特权 critic
使用，不得进入传统控制基线或部署 actor。所有演示必须在元数据中声明观测来源和证据
等级，禁止把预设目标坐标驱动的运动回放表述为视觉闭环。

## 里程碑总览

### M0：统一仿真闭环入口

建立唯一的数据流：

`MuJoCo RGB → 检测器 → 粗控制 → 内外环/PnP → 细控制 → 稳定验收 → 插入 → 记录`

统一粗对准观测、细对准观测、相机坐标系动作、插入授权、失败原因、轨迹指标和视频记录
接口。第一版必须能替换检测器，而不修改状态机或机器人适配器。

验收：同一入口可以运行传统检测器，控制路径不读取目标真值，输出逐帧观测、动作、阶段
和停止原因。

### M1：传统视觉粗对准

- 从 RGB 图像提取颜色、轮廓、圆或椭圆；
- 输出外环中心、尺度、置信度和拒绝原因；
- 使用 IBVS 将外环送入近场细对准区域；
- 支持目标丢失、保持、搜索和重新捕获；
- 从不同初始位姿重复运行。

验收：随机初始位姿下依靠图像进入细对准区域，保存世界视角、eye-in-hand 视角、轨迹
和失败原因。不得用仿真目标坐标生成控制动作。

### M2：YOLO 粗对准与对照

- 传统检测和 YOLO 实现相同的粗观测接口；
- YOLO 只负责远场发现和粗中心，不负责最终亚像素验收；
- 重新建立独立测试集，避免训练/验证图像重复；
- 比较光照、遮挡、尺度、背景变化、推理延迟和拒绝率。

### M3：内外环精对准

- 在近场 ROI 中分别拟合内环与外环；
- 计算亚像素中心、同心度、尺度和椭圆姿态；
- 通过 PnP/椭圆几何估计横向误差、悬停距离和通道轴角；
- 使用 PBVS 或 2.5D 视觉伺服输出有界任务空间微动作；
- 连续多帧全部通过才进入 `ALIGNED`，任一质量失败立即撤销授权。

### M4：对准后插入交接

只有 `insertion_handoff_ready=true` 才能启动冻结的传统插入控制器。插入期间继续监测
视觉质量、同轴度和安全裕度；目标丢失或质量门失败时停止。输出完整多视角视频、误差
曲线、最小间隙、壁接触和过插指标。

### M5：冻结传统基线

在不少于 500 个冻结测试回合中报告：

- 目标获取率、粗对准成功率、细对准成功率和全流程成功率；
- 最终像素、横向毫米、轴角和悬停距离误差；
- 对准时间、步数、路径比、折返、超调和抖动；
- 目标丢失、拒绝、最小间隙、壁接触和过插；
- 新初始位姿、光照、材质、相机噪声、戳卡姿态和标定扰动下的结果。

### M6：有界残差强化学习

在冻结传统基线后研究：

`最终动作 = 传统视觉伺服动作 + 有界 RL 残差`

RL 优化路径长度、折返、六轴耦合、阶段切换、超调、震荡和收敛时间。安全门、工作空间、
动作限幅、视觉质量门和最终插入授权保持为确定性约束。

### M7：双臂扩展

两臂分别通过 M0-M6 后，增加双方就绪屏障、空间避障、统一插入授权、同步进度控制和
联动停止。任一侧失效时不得继续单侧自主插入。

## 当前执行顺序

当前优先完成 M0-M4，得到第一段真正由相机 RGB 驱动的单臂“粗对准—细对准—安全
插入”仿真视频；随后冻结 M5，最后开展 M6 和 M7。

## A0：冻结接口和评价合同

- 定义粗对准输入、细对准输入、阶段状态和插入授权信号；
- 将三线合一拆成像素、毫米、角度、悬停距离和稳定帧数；
- 冻结传统方法、YOLO、传统视觉伺服和现有插入控制基线；
- 按完整初始位姿、场景和戳卡条件划分训练/验证/测试。

产出：机器可读阈值、阶段状态机、单元测试和评价表。

## A1：传统方法粗对准基线

- 评估颜色/轮廓/圆环/椭圆检测的目标获取范围；
- 从随机初始位姿生成有界趋近动作；
- 记录检测失败、视野丢失、路径长度和进入细对准区域的时间。

## A2：YOLO 粗对准

- 将 YOLO 作为远场戳卡外环检测器，不承担最终亚像素验收；
- 与传统检测在光照、遮挡、尺度和背景变化下比较；
- 输出检测框、中心、尺度、置信度及 OOD/拒绝状态。

## A3：内外环精对准

- 复用现有椭圆、内环、PnP、手眼标定和微步控制；
- 增加亚像素中心、内外环同心度、通道轴角和悬停距离；
- 使用连续多帧验收，禁止单帧直接触发插入。

## A4：传统全流程闭环

冻结“传统/YOLO 粗对准 + 几何细对准 + 传统插入”基线。至少报告：

- 目标获取率、粗对准成功率、细对准成功率和全流程成功率；
- 最终像素、横向毫米、轴角和悬停距离误差；
- 对准时间、步数、路径比、折返次数、峰值速度和抖动；
- 插入最小间隙、壁接触、过插和拒绝率。

## A5：强化学习对准优化

- actor 输入只使用部署可得的视觉、关节、TCP、阶段和动作历史；
- 输出有界任务空间残差或候选微动作，不直接输出无约束力矩；
- 奖励按词典序组织：安全/可见性 → 对准成功 → 时间/路径/平滑；
- 训练期可使用真值 critic，测试和部署 actor 禁止读取仿真真值。

比较组：

1. 纯传统两级对准；
2. 传统粗对准 + RL 细对准残差；
3. RL 粗对准路径优化 + 传统细对准；
4. 分阶段 RL 残差 + 硬验收门。

## A6：单臂全流程确认

只有在冻结测试集上满足以下条件才晋级：

- 不降低目标获取和最终对准成功率；
- 三线合一全部指标通过；
- 插入零新增硬失败；
- 至少稳定改善时间、路径、折返或抖动中的一项；
- 新初始位姿、光照和戳卡姿态上独立确认。

## A7：双臂扩展

双臂分别运行独立的 A1-A6 流程；新增协同仲裁、空间避障、双方就绪屏障、同步进度控制
和联动停止。单臂未通过时不启动本阶段。
