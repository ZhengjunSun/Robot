# 当前项目状态（2026-07-23）

## 正式主线

导师沟通后，项目核心已调整为插入前 eye-in-hand 视觉对准：

`目标获取 → 粗对准 → 细对准 → 三线合一稳定验收 → 传统插入`

既有插入代码保留为对准后的执行和最终安全验证模块。原插入风险强化学习结果作为历史
资产保留，不再决定项目首页主线。

## NIH/HRA 场景合同

- 眼球使用 NIH 3DPX-020963 / HRA Visible Human female right eye v1.2；
- 当前 NIH 页面公开下载仍指向 v1.2，未用页面文字中的 v1.3 说明冒充可下载资产；
- 眼球网格只承担解剖外观和视觉背景，不作为临床材料或碰撞精度证明；
- 戳卡使用当前冻结规格：外径 2.0 mm、内径 1.0 mm、壁长 2.5 mm、
  法兰外径 2.64 mm；
- 来源、许可证、哈希、归一化和组件审计位于
  `3d_modeling/external_assets/nih_hra_eye/`。

## 已完成

### M0：统一 RGB 闭环入口

- `real_3d_alignment/visual_loop.py` 统一检测、阶段判断、动作和逐步记录；
- `real_3d_alignment/mujoco_visual_env.py` 将控制侧 RGB 与评估侧真值隔离；
- `real_3d_alignment/staged_alignment.py` 提供 SEARCH、COARSE、FINE、ALIGNED
  阶段及 fail-closed 插入授权门；
- 控制器不读取戳卡目标真值。

### M1：传统视觉粗对准

- NIH 场景青色戳卡的 HSV/轮廓检测和有界 IBVS 已接入；
- 标准偏置约 8.60 mm，经 7 个 RGB 控制步进入细对准区；
- 末帧像素误差 6.262 px，评估侧横向误差 0.466 mm；
- 随机初始偏置 100 回合成功率 100%，平均 4.63 步，平均最终误差
  4.942 px / 0.856 mm，目标丢失 0 帧，平均折返 0 次。

### M2：YOLO 粗对准第一版

- 新增与传统检测相同的 `CoarseObservation` 适配器；
- 原项目旧 YOLO 权重在 NIH 新场景零样本测试中未检出目标，已记录为域差异，不复用
  其成绩；
- 使用特权几何分割生成 240/60/60 的训练、验证、独立测试集合成数据；
- YOLO11n CPU 训练 5 轮后验证已收敛并保留最佳权重；
- 独立合成测试：precision 0.9981、recall 1.0、mAP50 0.995、
  mAP50-95 0.8981；
- 标准在线闭环 7 步进入细对准区，最终 5.629 px / 0.448 mm；
- 同种子、同 10 个初始偏置的小样本对照中，传统和 YOLO 均为 10/10 成功；
  传统平均 4.5 步，YOLO 平均 4.4 步。

当前自动测试为 `18 passed`。

## 证据边界

当前只完成轻量笛卡尔 MuJoCo 模型中的 M1/M2 粗对准。YOLO 指标来自独立的合成
测试 split，不代表真实图像、不同仿真域或临床泛化；10 回合在线对照只验证接口和闭环
可运行，不是 M5 冻结基准。当前尚未完成：

- 完整六轴动力学控制；
- NIH 场景下的内外环/PnP 精对准闭环；
- 连续五帧 ALIGNED 最终验收；
- 对准失效撤销授权后的插入联动；
- 单臂完整“粗—细—插入”视频；
- 500 回合冻结评测、真实图像外部验证、强化学习和双臂协同。

## 下一步
