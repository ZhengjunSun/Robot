# 当前项目状态（2026-07-24）

## 项目方向

核心研究对象保持为插入前 eye-in-hand 视觉对准：

```text
目标获取
→ 传统方法 / YOLO 粗对准
→ 内外环细对准
→ 主动多视点孔轴确认
→ 连续稳定授权
→ 保留的传统轴向插入
```

强化学习不替代感知、几何质量门或插入安全门。只有传统单臂基线冻结后，才研究
“传统动作 + 有界 RL 残差”。

- M5.1：完成 5 个同种子修复冒烟，确认固定残差方案仍会错误授权；

## 当前阻塞

1. 法向误差中位数 1.068°仍略高于 1°目标，且当前样本不足以冻结 P95；
2. 协方差条件数阈值尚未完成统计标定；
3. 3 mm 主动视点的真实六轴微动作、回位和 ROI 可达性尚未接入 M4 视频全流程；
4. 旧 M5 正式 500 回合暂停，不能用更多失败样本替代感知修复；
5. M6 强化学习和 M7 双臂尚未开始。

## 下一步顺序

1. 扩大 M5.2 密集姿态网格并标定协方差拒绝阈值；
2. 把法向中位数压至 1°以内并验证 3 mm 视点全程 ROI；
3. 把主动三视点真实微动作接入统一六轴全流程；
4. 依次通过 5、37、100 回合分级验收；
5. 再运行正式 500 回合冻结；
6. 冻结后进入 M6，有界残差 RL 通过后才进入 M7 双臂。
