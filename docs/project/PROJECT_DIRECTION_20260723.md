# 项目方向确认：视觉对准为核心，安全插入为后置模块

## 1. 唯一主线

项目面向眼科微创手术戳卡入路，研究：

> 末端 eye-in-hand 摄像头引导的六轴机械臂分级视觉对准，以及通过对准验收后触发的
> 零内壁接触戳卡插入。

研究核心是插入前从初始位姿到戳卡正上方的自主视觉对准。既有插入控制保留为后置执行
模块，用于验证对准结果是否足以支持安全入路，不再把插入阶段残差 RL 作为第一主线。

## 2. 全流程

1. 搜索与目标获取：从初始视野定位目标戳卡。
2. YOLO/传统视觉粗对准：使用外环目标将戳卡送入近场精对准区域。
3. 内外环精对准：估计外环中心、内环中心、通道轴线和目标悬停距离。
4. 三线合一验收：摄像头光轴、戳卡外环轴线和内通道轴线连续多帧满足阈值。
5. 插入交接：视觉模块只发出授权信号，由冻结的传统插入控制器执行进给。
6. 运行时撤销：视觉质量、同轴度或安全条件失效时立即禁止或撤销插入授权。

## 3. “三线合一”的机器判据

单张二维图像中的三个中心重合，不足以证明三条空间轴线重合。正式验收至少包含：

- 摄像头主点与戳卡外环中心的像素误差；
- 戳卡内外环中心的同心度；
- PnP/椭圆几何估计的横向位移误差；
- 摄像头光轴与戳卡通道轴的夹角；
- 相机到戳卡平面的悬停距离误差；
- 重投影误差、检测置信度和连续多帧稳定性。

阈值必须由标定分辨率、手眼误差、TCP 重复性和后续插入安全间隙共同确定，不能只用
训练集像素误差决定。

## 4. 强化学习的正确位置

传统视觉和几何伺服是可解释基线及安全锚点。强化学习优先研究：

- 在粗对准阶段减少远场路径长度和折返；
- 学习六轴耦合下的任务空间残差微动作；
- 平滑粗对准到细对准的切换；
- 在细对准阶段降低超调、震荡和收敛时间；
- 在视觉不确定时选择保持、重观察或退回粗对准。

第一版不直接使用图像到关节力矩的无约束端到端控制。部署策略输出有界任务空间残差，
并受工作空间、关节、速度、碰撞和视觉质量门约束。

## 5. 两阶段研究

### 第一阶段：单臂

完成“目标获取—粗对准—细对准—稳定验收—传统插入”的完整闭环，并与纯传统控制比较
对准成功率、最终精度、耗时、路径效率、折返、抖动和插入零碰壁率。

### 第二阶段：双臂

两臂独立完成各自戳卡的分级视觉对准；协同仲裁器只在双方均稳定通过验收后统一授权
同步插入，并在任一侧失效时联动停止。双臂研究必须等待单臂正式通过。

## 6. 证据边界

现有代码已经包含真实图像圆环检测、PnP、手眼变换、质量门、微步建议、MuJoCo 场景和
插入控制资产。现有零碰壁结果主要属于仿真或代理证据；在真实标定、真实跟踪误差预算和
现场闭环完成前，不表述为临床级真机安全结论。
