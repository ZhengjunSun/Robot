# M5 冻结前就绪评测（2026-07-23）

## 定位

本轮建立了无视频批量入口 `run_m5_prefreeze_batch.py`，用于在进入正式 500 回合
冻结测试前发现传统基线的稳定性问题。

本结果不是正式 M5 冻结结论，原因是：

- 完整六轴 Meca500 闭环控制尚未接管当前笛卡尔 plant；
- 本条已被后续 M4 六轴闭环取代：当前使用 35° 斜置戳卡，主动旋转纠偏已完成；
- 还没有独立真实相机图像验证；
- 材质、遮挡、标定和可变戳卡姿态随机化尚未覆盖。

## 已固化场景

- NIH/HRA Visible Human female right eye v1.2；
- 眼球保持竖直，戳卡倾角 `35°`；
- 演示针长由 `108 mm` 缩短至 `36 mm`；
- 正式视频包含完整 Meca500 和摄像头/内环/外环三中心；
- 控制器不读取 MuJoCo 目标真值。

## 20 回合干净图像对照

随机条件：初始相机 XY 各轴在 `[-7, 7] mm`，固定 RGB 增益，无像素噪声。

这证明当前失败并非基本运动范围或随机初始平移导致。

## 20 回合图像扰动

相同随机种子，新增：

- RGB 增益 `[0.95, 1.05]`；
- 每帧独立高斯噪声标准差 `[0, 2]` 灰度级。

未去噪时：

| 指标 | 结果 |
| --- | ---: |
| 目标获取率 | 100% |
| 粗对准成功率 | 100% |
| 细对准成功率 | 75% |
| 全流程成功率 | 20% |
| 内壁接触 | 0 |

加入 3×3 高斯去噪后：

| 指标 | 结果 |
| --- | ---: |
| 目标获取率 | 100% |
| 粗对准成功率 | 100% |
| 细对准成功率 | 90% |
| 全流程成功率 | 50% |
| 平均对准步数 | 30.0 |
| 平均路径长度 | 14.721 mm |
| 平均折返次数 | 1.0 |
| 最小鲁棒间隙 | 0.214 mm |
| 内壁接触 | 0 |

5×5 去噪对照的全流程成功率降到 40%，说明过度空间平滑会损伤小尺寸内外环的
亚像素几何，因此没有保留。

## 结论

安全门按设计工作：所有失败都在视觉授权缺失、撤销或间隙不足时停止，没有新增
内壁接触。但 50% 的扰动全流程成功率不足以冻结 M5。

下一步优先级：

1. 为近场内外环估计增加不会产生明显延迟的时序稳健观测；
2. 分离“瞬时观测拒绝”和“持续失效停机”的安全时序合同；
3. 实现倾斜初始轴角下的主动旋转视觉伺服；
4. 迁移至完整六轴机械臂控制 plant；
5. 补齐真实图像和更完整域随机化后，再运行不少于 500 回合的正式冻结测试。
