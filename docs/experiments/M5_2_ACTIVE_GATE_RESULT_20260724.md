# M5.2 主动多视点状态门评测（2026-07-24）

## 本轮完成内容

状态机由可选的：

```text
SEARCH → COARSE → FINE → ALIGNED
```

扩展为启用 M5.2 合同时的：

```text
SEARCH → COARSE → FINE → OBSERVE_ACTIVE → ALIGNED
```

`OBSERVE_ACTIVE` 必须同时满足三视点可达、三帧外环检测完整、中心/轴角/悬停误差、
圆锥重投影残差和协方差条件数门。任意一项失败都会清零稳定计数并禁止
`insertion_handoff_ready`。

每组主动观测带单调递增序号；重复提交同一组缓存结果会返回
`active_multiview_observation_not_fresh`，不能累计连续稳定次数。

## 评测设计

入口：

```powershell
python run_m5_2_gate_evaluation.py
```

本轮包含：

- 6 个实际可达的已知姿态网格点；
- M5.1 的 5 个代表失败种子：47、7、22、139、79；
- 每例使用三个非共线安全视点；
- 旧单帧观测被刻意设置为“通过”，专门检查主动门能否撤销错误授权；
- MuJoCo 真值只决定离线“应授权/应拒绝”标签，不进入估计器或状态机。

## 结论边界

本轮证明主动状态门能在当前样本上阻断旧单帧错误授权，但尚未证明估计精度冻结：

- 最大法向误差 2.533°高于计划 P95 2°目标；
- 最大中心误差 0.322 mm 高于 0.20 mm 目标；
- 当前条件数阈值 `3e8` 是仿真冒烟暂定值，未完成概率标定；
- 本轮是近场感知/授权回放，不是完整 M5 机械臂对准与插入成功率。

因此 M6、正式 500 回合和双臂 M7 仍保持关闭。

## 下一步

1. 使用实际相机 FOV/主点状态和手眼协方差，不再把标定扰动全部吞入圆平面误差；
2. 对内外环轮廓联合加权，降低外环部分缺失和材质栅格造成的椭圆偏差；
3. 多初值保留圆平面双解，使用非共线视点重投影和法向连续性消歧；
4. 优化视点布局并加入 IK/ROI 可达性预检；
5. 运行更密集姿态网格，达到中位数 ≤1°、P95 ≤2°、错误授权 0；
6. 再把真实微动作 `OBSERVE_ACTIVE` 接入统一 Meca500 视频全流程。
