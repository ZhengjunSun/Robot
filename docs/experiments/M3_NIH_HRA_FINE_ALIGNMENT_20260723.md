# M3 NIH/HRA 内外环精对准基线（2026-07-23）

## 实现范围

本轮在 NIH/HRA 右眼场景中打通：

`RGB 粗对准 → 内外环椭圆 → 2.5D XYZ 微动作 → 连续 5 帧验收 → 插入授权`

控制器读取 eye-in-hand RGB、传统外环检测和内外环椭圆几何。MuJoCo 位姿真值只用于
世界视角和评价，不参与动作计算。

精观测包括：

- 摄像头光轴中心到戳卡外环中心的像素误差；
- 戳卡内外环同心度；
- 基于焦距、已知外环半径和图像尺度的横向/悬停距离误差；
- 外环椭圆长短轴比给出的轴角可观测量；
- 内外环椭圆拟合残差和几何比例质量门。

## 标准回合

初始相机横向偏置为 `(7, -5) mm`，输出双视角视频：

| 指标 | 最终结果 |
| --- | ---: |
| 总控制/确认步数 | 25 |
| 精中心误差 | 0.040 px |
| 内外环同心度 | 0.096 px |
| 视觉横向误差 | 0.003 mm |
| 视觉轴角误差 | 0.000° |
| 悬停距离误差 | 0.282 mm |
| 椭圆拟合误差 | 0.256 px |
| 连续稳定帧 | 5 |
| 插入授权 | true |

状态机在前 7 步完成粗对准，随后用最大 0.5 mm 横向、0.75 mm 悬停方向的有界微动作
收敛。全部指标通过后保持位置并连续确认 5 帧。

## 发现的失败模式

同样代码在 320×240 压力试验中为 0/30。进入近场后，传统 HSV 检测器可能从戳卡
切换到眼球上的相似青色区域，使状态机退回粗对准并形成反复切换。该失败结果被保留，
后续应使用以下一种或多种措施解决：

- YOLO 粗检测和传统内外环几何的交叉确认；
- ROI 时序锁定、尺度连续性和中心跳变拒绝；
- 与眼球背景颜色解耦的戳卡材质/真实图像检测特征；
- 分辨率低于可信阈值时直接拒绝，而不是继续运动。

## 证据边界

- 当前轻量 plant 只执行 X/Y/悬停距离平移；
- 相机与戳卡轴在场景中机械预对齐，轴角由图像估计并作为硬门，但尚未执行旋转纠偏；
- 小尺寸圆环的单帧轴角存在像素分辨率下限；
- 尚未接入真实六轴动力学、手眼标定扰动或插入控制器；
- `insertion_handoff_ready=true` 只表示软件质量门通过，不表示临床安全。
