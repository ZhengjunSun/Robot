# M4 NIH/HRA 单臂完整流程基线（2026-07-23）

## 数据流

```text
eye-in-hand RGB
  → 传统粗对准
  → 内外环精对准
  → 连续 5 帧 ALIGNED
  → insertion_handoff_ready
  → 独立器械进给轴
  → 每步视觉复验、解析间隙和 MuJoCo 接触记录
```

MuJoCo 目标真值不参与控制。控制侧读取 RGB、内外环几何和器械进给关节状态；世界
视角和实际接触仅用于评价。

## 场景修正

旧 M3 场景中的 50 mm 轴杆与相机刚性连接，调整相机悬停距离会让针尖提前越过戳卡
平面。本轮将相机对准滑台和器械进给轴分离：

- 相机完成粗/细对准后保持不动；
- 器械从标定的戳卡上方位置独立进给；
- 授权前进给轴严格保持零位；
- 圆形视觉戳卡不承担碰撞；四个隐藏的轴对齐墙体提供 MuJoCo 接触事件；
- 最终安全间隙仍按 1.0 mm 圆形内径、0.51 mm 器械外径和 2.5 mm 壁长解析计算。

## 标准结果

| 指标 | 结果 |
| --- | ---: |
| 粗—细对准授权 | 连续 5 帧通过 |
| 标定接近距离 | 10.0 mm |
| 壁内目标进给 | 2.5 mm |
| 单步进给上限 | 0.25 mm |
| 最小鲁棒间隙 | 0.222 mm |
| MuJoCo 壁接触步数 | 0 |
| 最大法向接触力 | 0 N |
| 过插 | 0 mm |
| 全流程停止原因 | target_insertion_depth_reached |
| 几何/接触合同 | 通过 |

插入期间每一步都重新运行粗检测、内外环精观测和 `StagedAlignmentGate`。目标丢失、
质量门失败或鲁棒间隙不满足阈值时，控制器返回零进给并停止。

## 证据边界

- 当前是轻量笛卡尔 plant，不是完整 Meca500 六轴动力学；
- 相机轴与戳卡轴在初始场景中平行，尚未主动执行旋转纠偏；
- 隐藏碰撞墙为轴对齐方形代理，不等同于完整圆柱内壁接触模型；
- NIH/HRA 眼球只承担视觉解剖背景；
- 该结果证明软件闭环和失效停止逻辑贯通，不构成真机或临床安全证明。
