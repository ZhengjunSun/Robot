1. 单帧内外环尺度、椭圆姿态和深度估计缺乏跨帧一致性检验。
2. 固定外环残差标定在随机几何和相机标定扰动下不能可靠代表真实同轴状态。
3. 插入接近戳卡入口时，视觉特征尺度和遮挡快速变化，单帧失败会立即撤权并结束，

## 3. M5.1 已建立的安全基础

新增 `TemporalFineObservationValidator`：

- 仅接收新鲜、质量门通过的观测；
- 对内外中心、横向、轴角和深度使用有限窗口中位数；
- 检查窗口内的变化范围；
- 缺帧或质量失败立即清空窗口，不用旧数据授权插入。

新增 `FailClosedReobservationGate`：

- 当前视觉无效时输出 `hold_and_reobserve`，机械臂必须保持原位；
- 新鲜视觉恢复后才能重新输出 `move`；
- 超过重观察预算输出 `abort`；
- 不允许在坏帧期间依靠宽限继续运动。

自动诊断脚本会从当前报告生成：

- `m5_checkpoint_diagnosis.json`
- `m5_1_replay_manifest.json`

复测清单以确定性规则从每个随机化层、每类结果中选择代表种子，后续 M5 与 M5.1
必须在相同种子和相同域参数下做配对比较。
