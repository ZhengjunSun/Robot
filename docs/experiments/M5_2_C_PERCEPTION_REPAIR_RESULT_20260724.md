# M5.2-C 标定、内外环与多解修复配对结果（2026-07-24）

## 修复内容

本轮针对 M5.2-B 随机化下中心和法向误差超限，加入三项可部署修复：

1. 显式使用经过标定的相机 FOV 和主点，不再固定假设 1280×960 图像中心和名义焦距；
2. 外环继续承担主要圆平面姿态约束，内环只以低权重中心/尺度约束参与联合拟合；
3. 从光轴和单目椭圆的两个法向候选进行多初值优化，再以多视点残差和条件数选择解。

内环不能与外环等权约束姿态。实验中，近圆内环的长轴方向受栅格影响跳变约
80°–150°；等权方案曾造成 10.75°法向误差。因此最终候选明确忽略内环不稳定的
长轴角度。

## 配对协议

入口：

```powershell
python run_m5_2_perception_repair_evaluation.py
```

旧版与候选版使用完全相同的 6 个姿态点和 5 个历史失败种子。MuJoCo 真值只用于离线
评分和“应授权/应拒绝”标签，不进入估计器。

旧版：

```text
名义内参 + 仅外环 + 单初值 + 1.5 mm 三视点
```

候选版：

```text
标定内参 + 外环姿态/内环中心尺度 + 多初值 + 3.0 mm 三视点
```

## 阶段结论

M5.2-C 候选可以晋级到更密集的姿态网格和实际六轴主动微动作工程验证，但还不能称为
冻结完成：

- 当前只有 11 个代表案例，样本量不足以冻结 P95；
- 法向误差中位数仍比目标高 0.068°；
- 3 mm 视点必须验证真实回位轨迹和全程 ROI；
- 内参输入属于部署标定信息，必须记录版本、协方差和新鲜度；
- M6、正式 500 回合和 M7 继续关闭。
