# M4 完整 Meca500 六轴统一闭环（2026-07-23）

## 结论

已在最新 NIH/HRA 竖直眼球、35° 倾斜戳卡和 36 mm 可见针具场景中打通：

```text
eye-in-hand RGB
  → 传统外环粗定位
  → 内外环椭圆中心与距离估计
  → 六轴平移 + 主动旋转视觉伺服
  → 连续 5 帧三线合一验收
  → 传统轴向插入
  → 插入中逐帧视觉与接触监测
```

左右视频画面由同一个 `Meca500VisualAlignmentPlant`、同一个 `MjModel/MjData`
逐帧渲染，不再使用独立运动学展示器。

## 关键修正

1. 原演示的右侧机械臂来自独立运动学展示器，左右画面只能阶段同步。现已删除该
   证据边界：两个相机共享完整六轴 plant。
2. 厚圆柱侧壁会把投影短轴撑大，使尚有 5–6° 误差时就看似圆形。视觉测量面改为
   薄青色环面，结构本体和碰撞代理仍保持完整厚度。
3. 插入时目标在画面中持续放大，因此像素阈值按近场分辨率调整；毫米横向误差、
   轴角、解析间隙和接触阈值不放宽。
